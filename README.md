# AWS Platform Engineering

Este repositório implementa uma plataforma AWS usando práticas modernas de
Platform Engineering, Terraform, CI/CD e automação.

## Componentes
- Terraform com backend remoto
- Módulos reutilizáveis
- Ambientes isolados (dev/prod)
- GitHub Actions
- Base para EKS, GitOps e Observabilidade

## Objetivo
Servir como fundação real de plataforma cloud.
